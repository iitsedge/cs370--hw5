README
======

This package includes the following files.

|-- Scheduler.cc [Responsible for implementing 3 scheduling algorithms (FCFS, SJFP, Priority) using the list of processes given to it within a .txt/.csv file.]
|-- Makefile [Provides way to compile, clean, and package program easily.]
|-- README.txt [This file]

To compile:
    make

To run:
    ./Scheduler <test_file>

For example;
    ./Scheduler test.csv
